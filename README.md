# Infinity Fitness Lab

A premium, dark-themed fitness website with interactive features.

## Overview

Infinity Fitness Lab is a modern, responsive website designed for a high-end gym. It features a glassmorphism aesthetic, neon accents, and a suite of client-side interactive tools.

## Features

-   **Interactive Booking System**: Browse the schedule and book classes (mock logic).
-   **Workout Plan Generator**: Create custom workout plans based on your goals.
-   **BMI Calculator**: Instant health insights.
-   **E-commerce Shop**: Browse products and manage a cart (sessionStorage).
-   **User Dashboard**: Track progress and view upcoming bookings.
-   **PWA Ready**: Includes manifest and service worker for offline capabilities.

## Tech Stack

-   **HTML5 & CSS3**: Semantic markup, CSS variables, Flexbox/Grid.
-   **Bootstrap 5**: Responsive layout and base components.
-   **Vanilla JavaScript (ES6+)**: No heavy frameworks, just clean modular code.
-   **AOS**: Animate On Scroll library for smooth reveals.

## Setup & Deployment

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/infinity-fitness-lab.git
    ```
2.  **Open Locally**:
    -   Simply open `index.html` in your browser.
    -   For best results (to avoid CORS issues with icons/fonts if any), use a local server like Live Server in VS Code.

## Customization

-   **Colors**: Edit root variables in `css/main.css`.
-   **Data**: Update `data/sample-data.json` (note: currently data is also hardcoded in HTML for standalone reliability).
-   **Images**: Replace placeholder images in `assets/images`.
